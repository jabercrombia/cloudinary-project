## Getting Started

cron scheduler

```sql
*/10  *  *  *  *  
 │     │  │  │  │
 │     │  │  │  └── Day of week (every)
 │     │  │  └──── Month (every)
 │     │  └─────── Day of month (every)
 │     └────────── Hour (every)
 └──────────────── Minute → every 10 minutes

```
